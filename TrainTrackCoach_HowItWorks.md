# Train Track - Client Manager: How It Works

## The basics

The Client Manager lets you track multiple clients from a single dashboard without any accounts, servers, or subscriptions. Each client gets their own private data slot saved in your browser.

---

## Adding a client

1. Click the green **Clients** button in the top right of the dashboard
2. Type the client's name in the **Add new client** field and hit Add
3. The dashboard switches to that client automatically - you are now in their slot
4. Upload their Excel file as normal - the data saves to their slot only

---

## Switching between clients

1. Click the **Clients** button - it shows the name of whoever you are currently viewing
2. Find the client you want in the list and click **Switch**
3. Their data loads instantly - the dashboard refreshes to show their progress
4. The current client's data is saved automatically before switching

---

## How data is stored

- Each client has their own isolated storage slot in your browser
- Loading one client's data never affects another client
- The **Default Client** slot is always available and cannot be deleted - this is the original single-client slot for when you only need one

---

## Updating a client's data

1. Switch to the client you want to update
2. Click **Update data** in the top right
3. Upload their new Excel file - their slot updates with the latest data
4. Switch to the next client and repeat

---

## Important limitations to know

**Browser only** - all data lives in this browser on this device. If you clear your browser storage, use a different browser, or switch to a different computer, the client slots will not be there.

**Storage limit** - most browsers allow around 5–10 MB of localStorage. Depending on how much data each client has this comfortably supports around 10–20 active clients before you may hit the limit.

**No sync** - there is no cloud backup. If a client's data is lost it is gone. Use the **Reports** button to print or save a PDF for each client regularly as a backup.

---

## Recommended workflow

| Task | How |
|------|-----|
| New client onboards | Add them in Client Manager, send them the Excel template |
| Weekly check-in | Client sends updated Excel → you switch to them → upload → review |
| Monthly review | Switch to client → click Reports → Generate Full Progress Report → save as PDF |
| End of program | Save final PDF report → delete client slot to free up storage |

---

## Deleting a client

Click the **✕** button on their card in the Client Manager. You will be asked to confirm. This permanently removes their data from the browser - export a report first if you want to keep a record.
